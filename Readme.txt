// ###################  Textbasiertes RPG - Die Rettung der Prinzessin #################

// Beschreibung:
// Der Spieler spielt einen Helden, der sich auf eine Reise begibt, um die Prinzessin aus den Fängen einer bösartigen Kreatur zu befreien.
// Auf dem Weg zur Prinzessin muss der Spieler eine Vielzahl von Hürden überwinden, die in Form von herausfordernden Quizzes auftreten.
// Der Spieler muss die Quizzes lösen, um die Prinzessin zu retten.
// Sollte der Spieler ein Quiz nicht lösen können, muss er sich einem Monster stellen, um das Quiz zu umgehen.


// Spielablauf:

// Der Spieler gibt seinen Namen ein und beginnt das Spiel mit 100 Lebenspunkten und Level 1.
// Auf dem Weg trifft der Spieler auf 8 Rätsel-Quizzes. Bei dem Quiz gibt es eine Frage und 4 Antwortmöglichkeiten. Der Spieler muss die richtige Antwort auswählen.
// Bei richtiger Antwort auf ein Quiz erhält der Spieler Punkte und kann zum nächsten Quiz weitergehen.
// Bei falscher Antwort auf ein Quiz verliert der Spieler Punkte.
// Er hat dann die Wahl: Entweder er kann das Quiz wiederholen und eine bessere Antwort finden oder er kann sich einem Monster stellen, um das Quiz zu umgehen.
// Der Spieler kann sich entscheiden, gegen das Monster zu kämpfen.
// Der Ausgang des Kampfes kann den Spieler entweder zum nächsten Quiz führen oder ihn dazu zwingen, das Quiz zu wiederholen und Punkte zu verlieren.
// Das Spiel gegen das Monster ist ein Glücksspiel. Der Spieler muss eine Zahl zwischen 1 und 9 erraten.
// Der Spieler hat 3 Versuche, um die richtige Zahl zu erraten. Wenn er die richtige Zahl errät, kann er zum nächsten Quiz weitergehen.
